# Work item schema

Frontmatter fields for `work/*.md` files in the lattice ecosystem.
Both `lattice-wiki` and `lattice-work` read from this path.

| Field | Type | Required | Notes |
|---|---|---|---|
| title | string | yes | |
| category | `work` | yes | |
| kind | enum | yes | bug \| tech-debt \| test-gap \| security \| perf \| feature \| initiative \| spike |
| summary | string | yes | |
| status | enum | yes | open \| accepted \| in-progress \| mitigated \| resolved \| wontfix \| superseded |
| severity | enum | no | Required for bug/security/perf kinds |
| effort | enum | no | trivial \| small \| medium \| large |
| blast_radius | enum | no | file \| package \| domain \| system |
| affects | list | no | Paths or packages this work touches |
| target | string | no | YYYY-QN or YYYY-MM |
| owner | string | no | Populate when in-progress |
| opened | date | yes | YYYY-MM-DD |
| updated | date | yes | YYYY-MM-DD |
| related_tickets | list | no | |
| related_prs | list | no | |
| resolved_in | string | no | Required when resolved |
| superseded_by | string | no | Required when superseded |
| mitigation | string | no | Required when mitigated |
| rationale | string | no | Required when wontfix |
| tags | list | no | |
