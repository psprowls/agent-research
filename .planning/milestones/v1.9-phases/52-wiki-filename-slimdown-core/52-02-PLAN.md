    ```python
    def _compute_collision_set(
        conn: sqlite3.Connection,
        admitted_kinds: frozenset[str],
        list_fns: dict[str, Callable],
    ) -> frozenset[str]:
        """Pre-pass that returns the set of URIs whose plain short stem collides.

        Iterates every admitted-kind node once, computes each node's *plain*
        short filename via `short_filename(uri, collision_set=frozenset(), ...)`
        (i.e. with an empty collision set so no suffix is added), groups by
        stem, and returns `frozenset(uri for stem in groups if len(group) > 1
        for uri in group)`.

        D-01 + D-02: extends to TestSuite kind-aware names by reading
        `suite_kind` from `node.attrs["suite_kind"]` and `pkg_for_suite` from
        `Path(node.attrs["path"]).parent.name` (or last segment fallback) when
        `kind == "test_suite"`.

        Pure-ish: reads the SQLite connection (read-only side-effect), but
        does not write or mutate global state.
        """
    ```

    Body algorithm:
    1. Build `stem_to_uris: dict[str, list[str]] = {}`.
    2. For `kind` in `sorted(admitted_kinds)`:
       - `list_fn = list_fns.get(kind)` — if None, `continue`.
       - For `node` in `list_fn(conn)`:
         - `uri = node.attrs.get("uri") if isinstance(node.attrs, dict) else None`. If falsy, `continue`.
         - If `kind == "test_suite"`:
           - `attrs = node.attrs if isinstance(node.attrs, dict) else {}`
           - `suite_kind = attrs.get("suite_kind") or None`  (treats `""` as `None`)
           - `suite_path = attrs.get("path")`
           - `pkg_for_suite = Path(suite_path).parent.name if suite_path else None`
           - If not `pkg_for_suite` (empty string): `pkg_for_suite = None`
           - `stem = short_filename(uri, frozenset(), suite_kind=suite_kind, pkg_for_suite=pkg_for_suite)`
         - Else:
           - `stem = short_filename(uri, frozenset())`
         - `stem_to_uris.setdefault(stem, []).append(uri)`
    3. Return `frozenset(uri for uris in stem_to_uris.values() if len(uris) > 1 for uri in uris)`.

    `Path` is already imported at line 185. `Callable` is already imported at line 186.
