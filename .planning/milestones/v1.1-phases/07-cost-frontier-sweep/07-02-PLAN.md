<objective>
Add per-role model_override surfaces on every command function the sweep runner needs to drive. Implements D-06's single-role-swap protocol on the agent side so the sweep runner (Plan 07-05) can pass exactly one candidate at a time while every other role uses its models.toml default.

Purpose: The sweep is meaningless without a way to swap one role's model in isolation. The current code only supports `librarian_model_override`; this plan adds equivalent surfaces for the other five in-scope roles.

Output: Four command files extended with override parameters, plus a focused unit-test file proving each override threads correctly to a ChatBedrockConverse with the right model_id.
</objective>
