<!-- WIKILINK_RE capture semantics (lint/common.py, line ~13) -->

```python
# Wikilinks: ``[[target]]``, ``[[target#anchor]]``, ``[[target|alias]]``.
# Inside markdown table cells the alias separator is escaped as ``\|`` so it
# doesn't collide with the cell delimiter — the lookahead ``(?!\\\|)`` stops
# the target at that escaped pipe so the alias group can consume ``\|alias``.
WIKILINK_RE = re.compile(r"\[\[((?:(?!\\\|)[^\]|#])+)(?:#[^\]|]*)?(?:\\?\|[^\]]*)?\]\]")
```

Group 1 captures the bare target. Anchor and alias suffixes are inside non-capturing groups in the same regex. The full match (`m.group(0)`) includes the full `[[...]]` form including any `#anchor` or `|alias`.

<!-- FENCED_CODE_RE + INLINE_CODE_RE (lint/common.py, lines ~30-32) -->

```python
FENCED_CODE_RE = re.compile(r"```.*?```", re.DOTALL)
INLINE_CODE_RE = re.compile(r"`[^`\n]*`")
```

Existing `strip_code(text)` helper applies both. But Phase 46 needs POSITION SPANS (for masking), not stripped text. So `rewrite_text` does its own span computation using these compiled regexes (no need to add new top-level constants).

<!-- New helper to add: indented_code_spans -->

CommonMark indented code blocks: a sequence of one or more lines starting with at least 4 spaces (or one tab), preceded by either a blank line or the start of the document. Continuation lines must also have the indent. The block ends when a non-indented non-blank line appears OR EOF.

Algorithm (one pass):
1. Walk lines of `text`, tracking line start position.
2. For each line, classify as: blank (whitespace only), indented (`^( {4}|\t)`), or other.
3. A code-block starts at a line whose previous line was blank (or it's line 1) AND which is indented.
4. The block extends through consecutive indented lines. A non-indented line ends the block.
5. Emit `(start_byte, end_byte)` spans — start is the start of the first indented line; end is the start of the first post-block non-indented line (or `len(text)` for EOF).

Empty list if no indented blocks found. The helper does NOT need to know about fenced blocks — the caller (link_rewriter) computes the union of fenced/inline/indented spans.

   def indented_code_spans(text: str) -> list[tuple[int, int]]:
       """Return spans of CommonMark indented code blocks in `text`.

       An indented code block is a run of one or more consecutive lines starting
       with at least 4 spaces or one tab, preceded by a blank line or the start
       of the document. The run ends at the first non-indented non-blank line
       (or EOF).

       Returns a list of (start_byte, end_byte) half-open spans, sorted ascending,
       non-overlapping. Empty input or text with no indented blocks returns [].

       Per CommonMark §4.4; used by `wiki_io.link_rewriter.rewrite_text` to mask
       indented code regions from wikilink rewriting.
       """
       if not text:
           return []
       spans: list[tuple[int, int]] = []
       lines: list[tuple[int, int, str]] = []  # (start, end, line_text)
       cursor = 0
       for line in text.splitlines(keepends=True):
           lines.append((cursor, cursor + len(line), line))
           cursor += len(line)

       def is_blank(s: str) -> bool:
           return s.strip() == ""

       def is_indented(s: str) -> bool:
           # Strip trailing newlines for the check but require 4-space or tab prefix.
           if s.startswith("\t"):
               return True
           # 4-or-more leading spaces; ignore lines that are blank
           if is_blank(s):
               return False
           return s.startswith("    ")

       i = 0
       n = len(lines)
       while i < n:
           start, end, ln = lines[i]
           prev_blank_or_start = (i == 0) or is_blank(lines[i - 1][2])
           if prev_blank_or_start and is_indented(ln):
               block_start = start
               block_end = end
               j = i + 1
               # Extend block through consecutive indented OR interior-blank lines whose
               # successor is also indented. Simpler rule: extend through indented lines
               # only; trailing blank lines are NOT part of the block.
               while j < n and is_indented(lines[j][2]):
                   block_end = lines[j][1]
                   j += 1
               spans.append((block_start, block_end))
               i = j
               continue
           i += 1
       return spans
   ```

   (The executor may simplify with regex `re.compile(r"(?m)^(?: {4}|\t)")` over `.finditer` + grouping consecutive line-spans; either approach is acceptable as long as the behavior contract holds.)

   def _code_region_spans(text: str) -> list[tuple[int, int]]:
       """Return sorted, merged spans covering fenced + inline + indented code regions.

       The union covers every position whose surrounding context is code.
       Wikilinks whose start position falls inside any returned span are SKIPPED
       by rewrite_text.
       """
       spans: list[tuple[int, int]] = []
       # Fenced
       for m in FENCED_CODE_RE.finditer(text):
           spans.append((m.start(), m.end()))
       # Inline — find on whole text; if an inline match falls inside a fenced span,
       # it's harmless (already covered).
       for m in INLINE_CODE_RE.finditer(text):
           spans.append((m.start(), m.end()))
       # Indented
       spans.extend(indented_code_spans(text))
       if not spans:
           return []
       spans.sort()
       # Merge overlapping/adjacent.
       merged: list[tuple[int, int]] = [spans[0]]
       for s, e in spans[1:]:
           ms, me = merged[-1]
           if s <= me:
               merged[-1] = (ms, max(me, e))
           else:
               merged.append((s, e))
       return merged


   def _is_inside_any_span(pos: int, spans: list[tuple[int, int]]) -> bool:
       """O(N) scan — N is tiny for typical markdown docs."""
       for s, e in spans:
           if s <= pos < e:
               return True
           if pos < s:
               # Spans are sorted; further spans are even later — short-circuit.
               return False
       return False


   def _rebuild_wikilink(original: str, old_target: str, new_slug: str) -> str:
       """Replace `old_target` with `new_slug` in the wikilink, preserving anchor + alias.

       WIKILINK_RE guarantees the captured target appears at the start of the
       bracketed content. `original.replace(old_target, new_slug, 1)` is sufficient
       because:
         - the target appears exactly once at the lead position
         - the `, 1` limits replacement to the first occurrence (defensive against
           an alias whose text happens to equal the target).
       """
       return original.replace(old_target, new_slug, 1)


   # ----------------------------------------------------------------------------
   # Public API
   # ----------------------------------------------------------------------------


   def rewrite_text(text: str, table: dict[str, str | None]) -> tuple[str, int]:
       """Rewrite old-layout wikilinks to new-layout slugs in `text`.

       Args:
           text: The markdown content of one file.
           table: Mapping from old-layout target string to new-layout slug.
               A `None` value means "discovered as inbound but unresolvable" —
               the wikilink is SKIPPED. A missing key also means SKIP.

       Returns:
           (new_text, rewrite_count). `rewrite_count` is the number of wikilinks
           actually rewritten (NOT the number of wikilink matches).

       Behavior:
           - Wikilinks inside fenced code blocks, inline code spans, or indented
             code blocks are SKIPPED (their bytes are preserved byte-identical).
           - Alias (`|alias`) and anchor (`#anchor`) suffixes are preserved.
           - Idempotent: a second call on already-rewritten text yields count == 0.

       See CONTEXT.md D-01 / D-02 for the decisions backing this design.
       """
       code_spans = _code_region_spans(text)
       parts: list[str] = []
       cursor = 0
       count = 0
       for m in WIKILINK_RE.finditer(text):
           if _is_inside_any_span(m.start(), code_spans):
               continue
           target = m.group(1).strip()
           new_slug = table.get(target)
           if new_slug is None:
               # Missing key OR explicit None — skip silently.
               continue
           # Splice text up to match start, then the rebuilt link.
           parts.append(text[cursor:m.start()])
           parts.append(_rebuild_wikilink(m.group(0), target, new_slug))
           cursor = m.end()
           count += 1
       parts.append(text[cursor:])
       return ("".join(parts), count)
   ```
