## Executive Summary

wiring (CLI → MCP → command kwarg → resolver → `wiki_dir` derivation) is end-to-end consistent and
covered by passing tests (workspace-wide `uv run pytest` reports **596 passed, 32 skipped, 0 failed**

mirror diverged. Because prompt-sources is what gets injected into the agent system prompts at
runtime, this means the agent describes an outdated classifier rule.

Broader investigation of the prompt-source mirror set revealed `wiki-schema.md` and
`ingest-workflow.md` are ALSO diverged from the plugin originals — predating Phase 25 (lattice-wiki →
graph-wiki rebrand drift). Phase 23 verified `scan-workflow.md` byte-identicality but did not
sweep the rest of the mirror set.
