Sketch 003 — Diff doc anatomy

What does raw/diffs/<pkg>-<from>..<to>.md contain when refresh writes it for ingest?

Example: packages/vault-io · 30058ee..4656429 · 82 commits, 47 files Reader: the ingestor sub-agent (LLM), then later a human auditing Cost lever: token budget of this doc directly drives ingest cost

A · Raw git diff. Wraps git log --stat + a real git diff in a markdown shell. Lossless. Mechanical to produce (no LLM in the loop). Big — ingest pays in tokens. Ingestor reasons over raw code changes.
