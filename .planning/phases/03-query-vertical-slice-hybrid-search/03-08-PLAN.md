<objective>
Close the SC-1 answer-quality gap diagnosed in `03-HUMAN-UAT.md`. Three failure modes have a single architectural cause: the librarian and synthesizer system prompts are too thin (~60 words each, no citation contract, no wikilink format spec, no no-invention rule, no stub-acknowledgment rule). This plan rewrites both prompt constants to encode the lattice-wiki librarian's workflow contract, and tightens `apply_guardrails` so unresolved wikilinks force a single synthesizer retry instead of being papered over with a warning footer.

Purpose: Recover the 3 prompt-fixable failure modes from `03-HUMAN-UAT.md` Gap 1 — fabricated file paths/symbols, missing `code-path:line` citations, malformed slug-only wikilinks — by replacing the prompt contract with one modeled on `~/.claude/plugins/cache/lattice/lattice-wiki/1.3.3/agents/librarian.md`. The fourth mode (no code-fallback when vault is thin) is intentionally out of scope here and handled by `03-09-PLAN.md`.

Output: Rewritten `LIBRARIAN_SYSTEM` and `SYNTHESIZER_SYSTEM` constants, a retry-on-unresolved-citation path in `apply_guardrails`, and unit tests that pin the new contract so it cannot silently regress.
</objective>
