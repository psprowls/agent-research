<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| prompt string → subprocess argv | User-supplied query string reaches the subprocess command list |
| plugin_dirs → subprocess argv | User-provided plugin directory paths reach the subprocess |
| baselines_dir / case_id → filesystem write | case_id used as filename component when writing snapshot |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-4-03 | Tampering | _build_cmd() subprocess construction | mitigate | cmd is always a list; subprocess.Popen(cmd, shell=False); prompt appended as a list element (cmd.append(prompt)) never via string interpolation; assert isinstance(cmd, list) guard at end of _build_cmd |
| T-4-01 | Tampering | BaselineRecorder cases JSON loading | mitigate | json.load() + validate case_id is str and query is str before use as filename/prompt; reject cases missing required keys |
| T-4-05 (new) | Tampering | case_id as filename component | mitigate | case_id sanitized before use in path: re.sub(r"[^a-zA-Z0-9._-]", "_", case_id) before constructing baselines_dir / f"{safe_case_id}.json" |
</threat_model>
