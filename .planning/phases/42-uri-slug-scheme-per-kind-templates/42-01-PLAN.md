```python
def file_uri(ctx: RepoContext, rel_path: str) -> str: ...
```

The 3 new builders Plan 02 adds (`package_family_uri`, `plugin_uri`, `dependency_uri`) are NOT available when this plan runs — Plan 02 runs in parallel. The Hypothesis strategy in this plan covers ONLY the 4 admitted kinds whose builders already exist (`repository`, `domain`, `package`, `test_suite`) plus inline-constructed URI strings (`f"package_family:{name}"`, `f"plugin:{name}"`, `f"dependency:{ecosystem}/{name}"`) for the 3 admitted kinds Plan 02 adds. After Plan 02 ships, Plan 03 may swap inline strings for the imported builders — that swap is not required for green tests.
