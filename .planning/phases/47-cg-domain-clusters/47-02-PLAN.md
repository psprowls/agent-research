   - Return `exit_codes.SUCCESS`.

5. `_render_human(result: cluster.ClusterResult) -> None` — prints to stdout. Output exactly the shape in D-21:
   ```

   Hub threshold: {hub_threshold:g}  ·  {n_packages_total} packages total

   ```
   followed by a "## Cross-cutting hubs ({len})" section listing each hub on its own line as:
   `  {name.ljust(name_w)}  — imported by {imported_by_count}/{n_packages_total} ({frac:.0%}) — connects clusters {csv-of-ids}`
   (use `", ".join(str(i) for i in connects_clusters)`), then for each cluster a section header `## Cluster {id}: {name} ({size} packages)` followed by `  - {member}` per member. If BOTH are empty AND `n_packages_total == 0`, print `No packages with import edges found.` to stderr (D-22) — but in the json-vs-human dispatch, only emit this when fmt is human (json prints the empty-shape JSON regardless).

   Width: `name_w = max(len(h.name) for h in hubs)` if hubs non-empty, else 0. Similar width pattern allowed for clusters; D-21 says "padding from longest name" — keep the implementation simple, no need for column alignment between sections.

The exact empty-output behavior:
- `fmt=json` with `clusters=()` and `cross_cutting=()`: still print the JSON object with empty arrays.
- `fmt=human` with `clusters=()` and `cross_cutting=()`: print `"No packages with import edges found."` to stderr; return SUCCESS.

Do NOT import or call `_format.render` — Phase 47 output is hierarchical and rendered inline (per RESEARCH.md and D-21).

Concrete helper:
```
def _seed_workspace(tmp_path, names, edges):
    import sqlite3
    from graph_io.schema import apply_schema
    from workspace_io.config import resolve
    # Create a git repo (apply_schema needs nothing git-related; resolve only needs --mode test)
    ws = resolve(tmp_path, require_manifest=False).workspace
    db_path = ws / "code.db"
    db_path.parent.mkdir(parents=True, exist_ok=True)
    conn = sqlite3.connect(db_path)
    try:
        apply_schema(conn)
        _seed_packages(conn, names)
        _seed_references(conn, edges)
    finally:
        conn.close()
    return ws
```
