<objective>
Fix the `_classify_dir` heuristic in `packages/wiki-io/src/wiki_io/detect_containers.py` so a top-level directory containing a mix of manifested package children and non-manifested siblings (or loose `.md` files) is classified as `package` instead of `ambiguous`. The fix makes the heuristic permissive (≥1 manifested child wins) per locked decision D-01.
