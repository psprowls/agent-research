<objective>
Implement SPARSER-01: extend the source-parser Python AST pass to populate
`_has_main_block` and `_has_importable_symbols` on the file-level SourceNode's
`attrs` dict. These attrs are consumed by `structural_nodes.emit` in Plan 03
to derive the `has_main` and `is_importable` File role flags (SPARSER-02).

Purpose: Decouple AST inspection from structural-node emission. AST work
belongs in source-parser (it already walks the tree); graph-io stays a pure
DB writer that reads the attrs out of `attrs_json`.

Output: source-parser produces the two attrs on every Python file; matching
fixture-driven tests cover positive + negative cases.
</objective>
