## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-01 | Information Disclosure | BedrockAccessDenied error message | mitigate | The error message MUST include the exact attempted ARN + `bedrock:InvokeModel` action so the user can fix IAM without guessing. | T-1-03 | Information Disclosure | models.toml | mitigate | `models.toml` contains ONLY model IDs and region — NO credentials. Acceptance criterion verifies no secrets are added: `! grep -E 'AWS_SECRET|aws_secret_access_key|sk-|sk_' cores/model-adapter/models.toml`. |
| T-1-05 | Spoofing | Inference-profile ARN typo | accept | A typo in `models.toml` would cause `ValidationException` from Bedrock, not a security issue. Acceptable risk for a personal project; researcher verified ARNs are correct. |
