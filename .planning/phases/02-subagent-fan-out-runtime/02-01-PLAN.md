| Boundary | Description |
|----------|-------------|
| `models.toml` → `loader.py` | TOML file is in-repo, version-controlled; trusted source. No untrusted input crosses this boundary. |
| `load_role_config(role)` → caller | Caller provides `role` string; KeyError on unknown role is the explicit failure mode (not an exception leak). |
