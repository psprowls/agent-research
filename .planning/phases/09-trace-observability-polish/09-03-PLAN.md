<objective>
Extend the `code-wiki-agent trace <file>` renderer to surface per-subagent cost using the existing per-record `cost_usd` field, grouped by `(role, model_id)`. This is OBS-05: replace the placeholder `Cost USD: (Phase 4)` line with a real cost rollup, sorted descending by total cost, formatted as `$0.000000`, with explicit `(+K unknown)` accounting for records whose `cost_usd` is `null`.

Purpose: Per D-07 (group by `(role, model_id)`), D-08 (rollup-only, not per-item), D-09 (`$0.000000` format, exclude nulls, append unknown count), D-10 (renderer must NOT import `eval_harness`), and D-15 (descending cost, alphabetical tie-break, `n/a` groups last).

Output: Extended `_aggregate_trace` + new rollup emission block in the `trace` command, plus a syrupy snapshot test locking the output for a mixed-model fan-out fixture (including at least one null-cost record).
</objective>
